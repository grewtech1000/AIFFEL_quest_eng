- [ ]  **1. 주어진 문제를 해결하는 완성된 코드가 제출되었나요?**

- [ ]  **2. 전체 코드에서 가장 핵심적이거나 가장 복잡하고 이해하기 어려운 부분에 작성된 
주석 또는 doc string을 보고 해당 코드가 잘 이해되었나요?**

    -  눈 랜드마크중 제일 잘 맞는 인덱스를 사용하였다.
    -  refined_x = int(x - w // 2)
       refined_y = int(y-h/2)
    - int() 함수를 사용해 소수점으로 인한 문제 해결.
